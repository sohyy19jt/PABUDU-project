Animation & Transition

Transition digunakan untuk efek perubahan halus, sedangkan animation untuk membuat gerakan berulang.

Contoh Transition:

button {
  background: blue;
  color: white;
  transition: background 0.3s;
}
button:hover {
  background: darkblue;
}

Hasil Transition:

Contoh Animation:

@keyframes gerak {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
.box {
  animation: gerak 2s infinite alternate;
}
