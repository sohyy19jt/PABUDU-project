Flexbox

Flexbox digunakan untuk mengatur tata letak dengan mudah secara horizontal maupun vertikal.

Contoh Flexbox:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
.box {
  background: lightcoral;
  padding: 20px;
  color: white;
  border-radius: 10px;
}
