Grid Layout

Grid Layout memudahkan dalam membuat tata letak dua dimensi (baris dan kolom).

Contoh Grid:

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}
.box {
  background: lightseagreen;
  color: white;
  padding: 20px;
  text-align: center;
}
