Info 1 — Pengenalan

Ini adalah panel pertama. Contoh ini menunjukkan bagaimana beberapa elemen HTML ditempatkan di lokasi yang sama dan diperlihatkan/ disembunyikan dengan visibility + transisi opacity.

Info 2 — Detail

Panel kedua bisa berisi teks, gambar, atau komponen interaktif lain (form, tabel, dsb). Kita menonaktifkan pointer events pada panel tersembunyi sehingga elemen di belakangnya tidak terganggu.

Info 3 — Tips

Jika ingin agar konten tetap mengambil ruang layout (tanpa absolute), kamu bisa menggunakan visibility tanpa posisi absolute — tapi elemen akan tetap menempati ruang sehingga layout bergeser. Untuk "sama persis" gunakan pendekatan absolute seperti contoh ini.
